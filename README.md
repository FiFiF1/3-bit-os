# 3 Bit OS

A small experimental operating system project intended for use with Codex.

## Goal

Build a tiny educational OS from the ground up, starting with clear architecture notes and small working pieces.

## Suggested starting milestones

1. Decide the target platform and boot path.
2. Create a minimal bootable image.
3. Print text to the screen.
4. Add a tiny command loop.
5. Add simple memory and file abstractions.

## Notes for Codex

Keep changes small and explain the reasoning behind low-level choices. Prefer simple, readable code over clever code.
